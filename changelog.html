v2.1.0 June 20, 2026

What's New in v2.1.0

New
  • Create Montages from Multiple Sources
    • Select multiple recordings, right-click and choose 'Create Montage from Selection…' to stitch a montage together from as many recordings as you'd like.
    • Drag and drop segments to reorder them.
  • Add Multiple Music Tracks to Montages
  • Game posters showing the matchup, outcome, and champion splash.
    • Posters generate in the background after upgrading, so your existing library catches up automatically.
    • KNOWN ISSUE: Chromas revert to the base skin. Will be patched in the future after more testing.
  • Initial macOS 27 Support
    • IMPORTANT: macOS 27 is currently in beta and subject to code changes. Patches will be issued as needed if something breaks.
Enhancements
  • Event timeline icons now use Riot's in-game art instead of emoji.
  • Objective names now match Riot's in-game names exactly.
  • Faster library loads when recordings live on slower storage like HDDs or network shares.
Bugfix
  • Fixed a bug that could clear your tags on launch.
Quality of Life
  • New installs default Recordings and Clips to the same directory. Existing setups are unchanged.
v2.0.0 April 30, 2026

The BIG one. Time for the fun stuff!

Important
  • Old recordings remain fully playable, but new features only apply to recordings made on 2.0 or later.
New
  • Tags and Notes
    • Create as many tags as you'd like, color code them, see them at a glance, sort by them, etc.
    • Every recording now includes a notes section where you can add and format notes about the matchup.
  • Theater Mode — make the video full screen within the app window, hiding the UI for more in-depth reviews without going full screen.
  • Clips Manager
    • Clips are now stored similar to recordings.
    • Apply tags, notes, etc just like recordings.
  • Create play montages straight from the app!
    • Clips has a new toggle at the top for creating montages.
    • Select which events you want to include in a montage, fine tune the start and end times on a per event basis.
    • Select from multiple transition styles to add some visual flair between clips.
    • Add your own music to set the tone.
Enhancements
  • Dashboard overhaul
    • New section at the top of the Dashboard tracking trends, averages, and intelligently detects who you've been struggling against.
    • Click any game row to get into all the details of the match.
    • Customize what stat pills show and where they show in the lineup — see only the information you believe is relevant for you.
    • Stats are tracked independently of the recordings; delete the recording when you're done with it but keep the stats.
  • Recordings management
    • Quickly select multiple recordings at once to make changes in bulk.
    • Lane matchup data is now tracked on Summoner's Rift.
    • Filter by wins/losses, champion played, matchup, game mode, tags, or any combination you'd like.
  • Timeline
    • More detailed event information — see who killed you, who you helped kill, or what team got that drake (including whether or not it was stolen).
    • Towers are now tracked.
    • Gold tracking added to Timeline.
    • Key level ups are now marked (2/3/6/11/16/18).
    • Timeline events spread out a bit when clumped together.
Quality of Life
  • Fleshed out storage retention settings — auto-delete by age, space consumed, or both, with an Apply button to instantly trim your library.
  • Black screen on tab-out replaced by a gray screen with an icon and some text, so YouTube no longer auto-trims empty black frames and desyncs audio on uploads.
